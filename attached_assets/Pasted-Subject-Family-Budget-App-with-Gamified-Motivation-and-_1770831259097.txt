Subject: Family Budget App with Gamified Motivation and Smart Shifting

Context:
Build a web application for a small family budget management. The app should be "vibe-coded" to be light, fun, colorful but professional, and highly encouraging.

Core Technical Requirements:

Language: The UI and all messages must be in Hebrew.

Tech Stack: Use React for frontend and a persistent database (Postgres/Replit DB) for the backend.

Authentication: Simple user login for family members to share one account.

RTL Support: Ensure the layout is Right-to-Left (Hebrew friendly).

Key Features:

Dynamic Dashboard: A "Fuel Gauge" or progress ring showing total monthly budget.

Category Management: Users can create/edit categories. If a category is over 100%, use a gentle, non-stressful color warning.

Smart Reset Date: A setting to choose the day of the month for budget reset.

"Borrow" Feature: Allow users to move funds from a surplus category to one that is over budget.

Fixed Expenses: Automated monthly deductions (Rent, etc.).

History: View and analyze spending from previous months.
